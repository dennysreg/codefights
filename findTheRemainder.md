Given two integers a and b, find the remainder of a when divided by b.

Example

For a = 5 and b = 3, the output should be
findTheRemainder(a, b) = 2.

Input/Output

[execution time limit] 4 seconds (py3)

[input] integer a

Guaranteed constraints:
2 ≤ a ≤ 20.

[input] integer b

Guaranteed constraints:
2 ≤ b ≤ 20.

[output] integer
